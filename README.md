# Java Internship Task 1 – Environment Setup & First Program

## Objective
Set up Java development environment and create a first executable Java program.

## Tools Used
- OpenJDK 17
- Command Prompt
- GitHub

## Steps Performed
1. Installed OpenJDK and configured JAVA_HOME.
2. Verified installation using `java -version` and `javac -version`.
3. Created Java project structure manually.
4. Wrote a Java program with `main()` method.
5. Compiled program using `javac`.
6. Executed program using `java` command.
7. Passed command-line arguments.

- `Day1.java` – Source code
